# Go CLI Login System

Simple command-line login and registration system written in Go.

## Overview

This project implements a basic authentication flow in the terminal.
Users can register, log in, and access a simple system area.

## Features

* User registration
* Login with username and password
* Age validation (18+)
* Basic input validation
* In-memory user storage

## What I Learned

* Structs for data organization
* Slices to store multiple users
* Loops and conditionals
* Basic error handling
* Function organization

## How to Run

1. Install Go
2. Clone the repository
3. Run:

```bash id="hpn3tu"
go run main.go
```

## Usage

* Type 1 to register
* Type 2 to log in
* Type 3 to exit

## Limitations

* Data is not persistent
* No password encryption
* Basic validation only
